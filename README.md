# Git Introduction — A Tutorial to Git Fundamentals

This repo contains a comprehensive introduction to Git. This tutorial 
is not for complete beginners — it is for users who already know how 
to clone a repo and push changes to GitHub, but want a more thorough 
understanding of Git workflows and best practices.

> 📝 Note: Git and GitHub are different products. Git is open source 
> version control software that runs on your machine. GitHub is a cloud 
> hosting service built on top of Git.

In this tutorial, we use a Python BMI calculator as a practical example 
so that learners can see real code changes and their effect on git history.

## Prerequisites

- A GitHub account
- Comfortable using the command line
- Know how to clone a repo and push basic changes
